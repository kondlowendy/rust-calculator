Project Overview Rust Calculator 

Project Overview 

This project is a simple command-line calculator written in Rust. It was developed as part of the Postgraduate Diploma in Software Engineering. The goal of the project was to practice basic software engineering principles such as writing structured code, handling user input, managing errors, and using version control with GitHub. 

The calculator demonstrates how Rust can be used to build small but functional programs while keeping the code organized and easy to understand. 

Features 

Performs basic arithmetic operations: 

Addition (+) 

Subtraction (-) 

Multiplication (*) 

Division (/) 

Handles division by zero without crashing the program 

Provides a simple command-line interface for user input 

Uses Git and GitHub to track changes and manage commits 

Requirements 

To run this project you need: 

Rust (latest stable version) 

Ubuntu/WSL or a Linux environment 

How to Run 

Open the Ubuntu terminal (WSL). 

Navigate to the project folder: 

cd /mnt/c/Users/User/rust-calculator 

Run the program using Cargo: 

cargo run 

Follow the prompts in the terminal: 

Enter the first number 

Enter the second number 

Choose the operation (+, -, *, /) 

Example Run 

Simple Rust Calculator 
Enter first number: 10 
Enter second number: 5 
Choose operation (+, -, *, /): + 
Result: 15
